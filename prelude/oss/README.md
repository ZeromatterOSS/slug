# Kuro Prelude

This repo contains a copy of the Kuro Prelude, which is often included as a
submodule with a Kuro project. To obtain a copy of this repo, and set up other
details of a Kuro, you should usually run `kuro init --git`. Most information
can be found on the main
[Kuro GitHub project](https://github.com/facebook/kuro).

Pull requests and issues should be raised at
[facebook/kuro](https://github.com/facebook/kuro) as that project is more
closely monitored and contains CI checks.
